# GrantAI — Backend (Spring Boot)

Spring Boot 3 · Java 21 · PostgreSQL · Redis · JWT Auth

## Overview

REST API server handling:
- User authentication (JWT)
- Grant CRUD & search
- Application pipeline management
- Webhook integrations

## Setup

```bash
# Requires Java 21 and a running PostgreSQL instance
./mvnw spring-boot:run
```

API available at `http://localhost:8080`  
Swagger UI at `http://localhost:8080/swagger-ui.html`

> **Note:** Spring Boot project will be initialized in the next prompt.
